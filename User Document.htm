1.    Welcome Page

 

The page above shows the Main page of the application. It contains 2 buttons.

·         When user press on Dark button, it will change to night mode

·          When user press on Light button, it will change to light mode.

·         When user swipe left, it will load list page

 Please read the whole user document to understand how the application works.

2.    List Page

 

2.1 Search Contact

Users need to enter in text field in order to search for a specify contact

 

2.3 Delete

Users need to do long press in each contact and drag it to Delete Button, it will delete that contact

 

2.3 Sort

If user want to sort alphabet in ascending or descending, press A-Z button, it will sort according to your needs

3.    Detail Page

When users want to see further detail, users need to press on each contact, it will load Detail Page and display according to specific contact

4.    Add Page

Users need to Press on Add Button in the List Page and it will load to Add Page

Users need to fill all textboxes in order to save a contact.

 

5.    Edit Page

Users need to swipe left on each contact and press on Edit Button, it will load Edit Page

Users need to fill all textboxes in order to edit a contact. Press Save Button and it will load list Page back.
